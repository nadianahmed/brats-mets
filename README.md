# BraTS-METS

This project is an attempt at coming up with rules that can aid physicians in finding brain tumours.

## Getting the datasets

Make sure to follow the [instructions](Datasets/Instructions.md) to load the data.

## Installation

Install all the required Python libraries using:

```
pip install -r requirements.txt
```

## Usage

Run the main program

```
python main.py
```
